Easily hide all admin notices with a single plugin activation.

== Description ==

This plugin allows you to quickly and easily hide all admin notices in your WordPress dashboard with a single activation. Once activated, all admin notices will be hidden from view. There is no need to configure any settings, and you can easily deactivate the plugin to restore normal admin notice behavior.

== Installation ==

Upload the plugin files to the /wp-content/plugins/hide-admin-notices-easily directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.
